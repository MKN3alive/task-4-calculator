# Задача №4. Калькулятор

**Задача**

Реализовать интерактивны калькулятор в вещественных числах, поддерживающий следующие операции:

- сложение, умножение, вычитание, деление, степень
- скобки
- унарный минус
- операцию взятия предыдущего результата _

Также калькулятор должен:

- учитывать естественную очередность операций
- поддрживать числа inf, +inf, -inf, nan
- корректно обрабатывать деление на ноль и операции с бесконечностями.

Пользоваться eval нельзя.

Задание выполнено без создания виртуального окружения и не имеет внешних зависимостей.
